Introduction

HTML, or HyperText Markup Language, is the core building block of the web. It structures and formats content for websites.

The language is based on "tags," which define elements like paragraphs, links, headings, and more.

HTML Basics

Every HTML document begins with a doctype declaration to tell browsers it’s an HTML5 document:

<!DOCTYPE html>

Then it includes the following key elements:

  • <html>: Root element
  • <head>: Metadata for the document
  • <body>: Content visible to the user

An example of a basic HTML document:

<html>
<head></head>
<body></body>
</html>
HTML Tags

Tags are used to define elements in HTML. They usually come in pairs:

<p>This is a paragraph</p>

Tags can also have attributes that provide additional information, such as:

<a href="https://example.com">Click Here</a>

Common HTML tags include:

  • <h1> to <h6>: Headings
  • <p>: Paragraph
  • <img>: Images
Forms and Inputs

Forms are essential for gathering user input. The key form-related tags are:

  • <form>: Defines a form
  • <input>: Input fields
  • <button>: Buttons for submitting data

Here’s an example form:

<form>
<input type="text">
<button>Submit</button>
</form>

Forms can include other input types like:

  • <textarea>: Multiline text input
  • <select>: Dropdown menus
Best Practices

Clean, semantic HTML improves accessibility and readability. Key practices include:

  • Use descriptive tags like <header>, <footer>, and <article>.
  • Keep your code well-indented and organized.
  • Use meaningful attribute values, like alt for images.

Consider this example:

<img src="image.jpg" alt="Description of the image">

Good semantic HTML ensures websites are both user-friendly and machine-readable.
